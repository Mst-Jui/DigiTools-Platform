 # 🚀 DigiTools 
📌 **Project Overview**  
DigiTools is a modern web application designed to supercharge your digital workflow. It provides access to premium AI tools, design assets, templates, and productivity solutions—all in one place to help users work faster and smarter.  

--- 


## 🛠️ Technologies Used  
- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🌼 DaisyUI  
- 🔔 React Toastify  
- 📦 JavaScript (ES6+)  

---

## ✨ Key Features  
- ✅ Interactive pricing plans with dynamic selection  
- 🛒 Add to cart system with real-time updates  
- 🔔 Beautiful toast notifications for user actions  

---

## 📦 Dependencies  
- react  
- react-dom  
- react-toastify  
- tailwindcss  
- daisyui

## ⚙️ Run Locally  

Follow these steps to run the project on your local machine:

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/DigiTools.git

# Go to the project folder
cd DigiTools

# Install dependencies
npm install

# Run the development server
npm run dev
